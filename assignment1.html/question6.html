Ordered list of HTML Tags

  1. a tag

    This is called ancor tag, it is used to deffine hyperlink in the web page

  2. b tag

    This is called bold tag , it is used to make your text bold in the web page

  3. body tag

    This tag is to define the document's body

  4. br tag

    This tag is used to break line

  5. button tag

    It is used to define clickable button

  6. dl tag

    It is used to define description list

  7. Form tag

    It is used to define form for user input

  8. img tag

    It is used to define an image

  9. head tag

    It is used to define information about the document

  10. ol tag

    It is used to define ordered list

  11. ul tag

    It is used to define unordered list

  12. li tag

    It is used to define list items

  13. tabe tag

    It is used to define table in the web page

  14. nav tag

    It is used to define navigation list

  15. noscript tag

    This tag is used to define an alternate content for users that do not support clint-side scripts

  16. p tag

    It is used to define a paragraph

  17. param tag

    It is used to define parameter for an object

  18. section tag

    It is used to define a section in a documents
